### :man_technologist: О проекте :

Данный проект (далее просто Проект) представляет собой базовый (минимальный) сервис для работы с базой сотрудников.

#### Задача:
* Реализация работы с БД при помощи подхода ORM.
* Реализация примера использования "адаптивной" верстки web-страницы при помощи JS.
* Использование CSS для оформления страницы.

 
#### Техническая часть:
* Основные зависимости находятся в файле requirements.txt.
* Бэкенд построен на Flask.
* Основные страницы - Главная, Ввод данных сотрудника и Поиск по сотрудникам.

#### Описание запросов:

| Функция                             | Идентификатор        | Комментарий                                                             | 
|-------------------------------------|----------------------|-------------------------------------------------------------------------| 
| Главная страница                    | /                    | Для перехода к следующему функционалу                                   | 
| Ввод нового сотрудника              | /add                 | Маршрут для страницы добавления сотрудника (GET-запрос)                 |  
| Просмотр списка пользователей       | /search              | Маршрут для страницы поиска (GET-запрос)                                | 
| Создание склада                     | /search_results      | Маршрут для обработки результатов поиска (POST-запрос, принимает JSON)  | 
| Просмотр складов                    | /add_employee        | Маршрут для обработки формы добавления сотрудника (POST-запрос)         | 

#### Запуск проекта:
Для корректной работы необходимо установить зависимости pip install -r requirements.txt
Для тестирования проекта достаточно выполнить python app.py.
